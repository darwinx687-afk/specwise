# v0.3 Scope Decision Record

Status: Proposed, pending owner review before Phase 26/27 implementation.

这是 v0.3 范围决策记录，不代表已经开始实现。

## Decision

v0.3 should prioritize reliability, review usability, handoff readability, deterministic workflow inference, and public repo maintainability before real AI provider calls.

## Accepted Baseline

- GitHub Actions CI planning / optional CI
- Review/apply-plan usability polish
- Handoff pack readability polish
- Deterministic workflow inference polish
- Documentation consolidation

## Deferred

- Real AI provider runtime
- OCR / vision
- Web UI
- Agent calls
- Auto-apply patches
- Final spec-pack generation
- npm publish

## Rationale

SpecWise v0.2 improved examples, deterministic drafts, CSV inference, CLI/DX, and first-time docs. The next safest step is to improve reliability, review readability, and maintainability before adding high-risk runtime capabilities.

There is no public issue feedback yet. This decision is therefore a proposed baseline, not a feedback-confirmed roadmap.

## Safety Notes

- This record does not start v0.3 implementation.
- This record does not authorize real AI provider calls.
- This record does not authorize coding agent calls.
- This record does not authorize npm publication.
- This record does not authorize auto-apply behavior or final spec-pack generation.

## Review Point

Review this decision before Phase 26 and again before any Phase 27 implementation phase.
