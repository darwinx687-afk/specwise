# Start Here v0.2

Status: Phase 27E consolidated first-time navigation map.

Use this page when you are opening SpecWise for the first time and want the shortest safe reading path.

SpecWise v0.1/v0.2 baseline commands are local-first. They do not call real AI providers, do not call coding agents, and do not generate application code.

Do not start by reading every document. Pick the section below that matches your job.

For current-vs-historical document status, see [Docs Status Map v0.3](docs-status-map-v0.3.md).
