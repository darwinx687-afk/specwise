## 1. GitHub Repo Intro

SpecWise turns messy legacy business materials into reviewable, evidence-based spec packs for AI coding agents.

SpecWise v0.1.0-preview.0 is a local-first, dependency-free CLI preview for preparing legacy-system rebuild materials before coding starts. It helps organize facts, assumptions, open questions, evidence, permissions, workflows, acceptance criteria, and handoff boundaries.

Current boundary: no real AI provider calls, no coding agent calls, no application code generation, and no final spec-pack generation. Outputs stay review-required.

## 7. GitHub Release Share Copy

SpecWise v0.1.0-preview.0 is available as a GitHub pre-release.

SpecWise turns messy legacy business materials into reviewable, evidence-based spec packs for AI coding agents.

This release focuses on local CLI workflow, schemas, validation, deterministic draft generation, offline patch preview, human review reports, manual apply plans, and context-only handoff skeletons.

Boundaries remain explicit: no real AI provider calls, no prompt runner, no OCR or vision, no Web UI, no coding agent calls, no application code generation, no final spec-pack generation, and no auto-apply patch behavior.
