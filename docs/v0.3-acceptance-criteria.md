# v0.3 Acceptance Criteria

Status: Phase 27C acceptance checklist with handoff readability completion note.

This checklist defines acceptance expectations for later Phase 27 work. It does not start implementation.

## CI / Reliability

- [x] CI plan is documented.
- [x] CI implementation has separate owner approval before any workflow file is added.
- [x] CI workflow avoids npm publish and releases.
- [x] CI workflow avoids secrets and external services.
- [x] Local validation commands remain documented and runnable outside CI.
- [x] CI safety documentation exists.

## Review / Apply-plan UX

- [x] Review report is easier to scan.
- [x] Review decision summaries are clearer.
- [x] Follow-up questions remain visible.
- [x] Manual apply plan still does not auto-apply.
- [x] Blocked items remain visible.
- [x] No final spec-pack generation is introduced.
- [x] Dependency-free `review:quality` smoke check covers the local review/apply flow.

## Handoff Readability

- [x] Handoff pack still validates.
- [x] Open questions remain prominent.
- [x] Root README summary is easier to scan.
- [x] Agent-specific files still say they are not instructions to implement.
- [x] Handoff artifacts remain context-only.
- [x] No agent calls or task generation are introduced.
- [x] Dependency-free `handoff:quality` smoke check covers the local handoff flow.

## Workflow Inference

- [ ] Existing examples still validate.
- [ ] Deterministic draft checks still pass.
- [ ] Uncertain transitions become questions.
- [ ] Ambiguous workflow notes do not become confirmed facts.
- [ ] No AI calls are introduced.
- [ ] Draft output remains review-required.

## Docs Consolidation

- [ ] First-time path remains clear.
- [ ] Known limitations remain visible.
- [ ] Privacy and safety docs remain linked.
- [ ] Older planning docs remain findable as historical context.
- [ ] Docs do not claim production readiness.
- [ ] Docs do not imply real AI extraction is implemented.

## Release Readiness

- [ ] All selected Phase 27 workstreams are complete or explicitly deferred.
- [ ] `node --check` passes.
- [ ] `cli:dx` passes.
- [ ] `field:quality` passes.
- [ ] `draft:quality` passes.
- [ ] `review:quality` passes.
- [ ] `handoff:quality` passes.
- [ ] `npm test` passes.
- [ ] `npm run smoke` passes.
- [ ] Hygiene scan passes.
- [ ] No new dependency is added without separate approval.
- [ ] Package remains private unless separately approved.
- [ ] No new tag is created without owner approval.
- [ ] No GitHub release is created without owner approval.
- [ ] No npm publish occurs.
