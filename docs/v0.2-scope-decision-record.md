# v0.2 Scope Decision Record

Status: Proposed, pending public feedback.

## Decision

v0.2 should prioritize DX, deterministic extraction quality, and example depth before real AI provider calls.

## Reason

SpecWise's trust depends on evidence-first, review-required behavior. Improving deterministic quality and examples is safer and more useful than adding provider calls too early.

## Accepted

- More realistic examples
- Deterministic heuristic polish
- CSV/XLSX inference polish
- CLI/DX polish
- Docs simplification

## Deferred

- Real AI provider runtime
- OCR/vision
- Web UI
- Agent calls
- npm publish

## Status

Proposed, pending public feedback.

## Notes

This record does not start v0.2 implementation. It records the Phase 20 baseline so Phase 21 can plan small, reviewable work without changing the v0.1 preview boundary.
