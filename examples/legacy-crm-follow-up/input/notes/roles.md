# Roles

## System Admin

- Can configure roles and permission settings.
- Can view system configuration.
- Should not be treated as normal sales owner.

## Sales Director

- Can view all customers.
- Likely can export customer and follow-up records.
- Can review manager-level reports.

## Sales Manager

- Can view team customers.
- Can review follow-up records submitted by team reps.
- Cross-team visibility is unclear.
- Export permission is unclear.

## Sales Rep

- Can view assigned customers.
- Can create follow-up records.
- Can submit follow-up records.
- Whether they can edit submitted records is unclear.
- Whether they can reassign customers is unclear.

## Viewer

- Can view reports only.
- Should not edit customer or follow-up records.

## Unclear Permissions

- Export permission by role.
- Cross-team customer visibility.
- Who can mark customer inactive.
- Whether reassignment requires manager approval.
