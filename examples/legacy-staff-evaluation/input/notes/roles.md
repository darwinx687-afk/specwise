# Roles and Access Notes

## President / 总裁
- Can view all company-level, department-level, group-level, and employee-level records.
- May export reports, but this should be confirmed.
- Should not necessarily edit every record directly.

## Company Admin / 公司管理员
- Can manage records within their own company.
- Can import employee data for their company.
- May configure company-level roles.

## Department Manager / 部门主管
- Usually reviews evaluation records within their own department.
- It is unclear whether cross-department view access is allowed.
- It is unclear whether department managers can export all department records.

## Group Leader / 组长
- Can submit evaluation records for group members.
- May edit draft records before submission.
- It is unclear whether submitted records can still be edited.

## Evaluator / 评估专员
- Can create evaluation records.
- Can submit records for review.
- Review or approval permission is unclear.

## Viewer / 只读账号
- Can view assigned records.
- Should not create, edit, approve, import, export, or configure permissions.
